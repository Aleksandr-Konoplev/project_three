# Project Three

Проект на **Django**, который на данный момент выводит две страницы: 

- **Главная** (`home`)  
- **Контакты** (`contacts`)

---

## Технологии

- Python 3.13  
- Django  
- Bootstrap  
- SQLite  

---

## Установка и запуск

1. Установить зависимости через **Poetry**:
```bash
poetry install
```

2. Запустить сервер разработки:
```bash
python manage.py runserver
```

3. Открыть в браузере:
```bash
http://127.0.0.1:8000/
```
